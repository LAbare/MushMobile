MushMobile

Veuillez lire la page entière avant d'installer le script !

Firefox est vivement recommandé pour ce script, puisque le Flash reste nécessaire pour avoir accès à toutes les fonctions du jeu. Ce script modifie l'interface et permet une redondance de certaines fonctions sans Flash ; cependant, la minimap et la liste et la visée des Hunters ne sont pas supportées : il faut toujours passer par le jeu Flash pour y avoir accès.

Activation manuelle : ajoutez ce lien dans vos favoris (comment faire ?) : MushMobile.

Lien direct Tampermonkey TEST : ici

Le script doit être activé manuellement à chaque fois que vous rouvrez le navigateur. Heureusement, c'est simple : allez sur Mush, puis tapez « MushMobile » dans la barre d'adresse et cliquez sur le favori que vous avez enregistré qui sera proposé par votre navigateur. Vous pouvez recharger la page sans avoir à refaire cette procédure en utilisant le bouton . Le bouton permet de reconstruire l'interface mobile si jamais elle « casse » (après certaines actions).

Activer le script sur Mush :

Installer Flash

Requis : Android 4.3 maximum. (Voir Paramètres → À propos du téléphone → Version d'Android)

  1. Activez l'installation d'applications externes (Paramètres → Sécurité → Sources inconnues coché) ;
  2. Installez Flash Player (une fois le téléchargement terminé, ouvrez la barre de notifications et cliquez sur le fichier pour lancer directement l'installation) ;
  3. Installez un navigateur qui sait gérer le Flash (Firefox, Dolphin Browser).
