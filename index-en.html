Presentation

Small(Mush) is a script for Mush coded by LAbare, thanks to which you may play your favourite game on your phone. Yes, that's right. Moreover, if your phone does not support Flash, the script is able to emulate the basic functionalities of the game. Neat, isn't it?

However, I cannot stress enough that this is a script. It is not a mobile app for Mush. It is coded by a mere player, without the Motion Twin having anything to do with it; the game interface has to be slightly modified, and all bugs that occur while the script is active have to be reported to me, not on the forums. The main function of this script is to fit the large game in the small size of your phone. For any question, suggestion, any chocolate or love you may want to give me, send me a message.

Use

In order for the game to fit on your phone, Small(Mush) divides it into five tabs.

Top left: these buttons will refresh the game, and the help button will display any tooltip you want, even on added elements.
Top right: this button opens the Small(Mush) parameters menu. Try it, there are useful options!
The tabs bar gives you access to the five tabs: Yourself, Ship, Room, Chat and Module (that is the Flash game and the terminals you access), plus the vending machine.

Installation and activation

The script may be automatically activated for Firefox and Dolphin Browser on Android ; for some other browsers, such as Chrome, you may activate it manually each time you play Mush.

Firefox (Android)

  1. Install the Small(Mush) addon.

This Firefox addon is now outdated and not supported anymore. Try at your own risk… if you even manage to install it.

Dolphin Browser (Android)

  1. Install the Tampermonkey for Dolphin Browser addon from the Play Store.
  2. Install the script by clicking on this link, then « Install » in the next window.

Other browers (manual activation)

Warning: this may not work depending on the browser you use.

  1. Click on this link and add the page to your bookmarks. (Only the URL changes, not the page.)
  2. Edit the bookmark you just created (long-press on it from the Bookmarks menu of your browser).
  3. In the URL field, erase the hash (#) and all that is before it. You should get some code beginning with javascript:.
  4. Once you're on Mush and the page is loaded, enter "Small(Mush)" in your browser address bar. It should display some links, and among them your bookmark; click it. You may also just open the bookmark from your Bookmarks folder, but that may not work.

This works well with Chrome, as well as with Safari for iPhone owners.

Install Flash

Note : A few years have passed since I first wrote this tutorial; it's now almost guaranteed that you won't be able to install or use Flash on mobile.

Although not necessary for this script, Flash will allow you to fully experience Mush.

Required: Android 4.3 maximum. (See Settings → About my phone → Android version)

  1. Allow the installation of third-party apps (Settings → Applications or Security → Unknown sources checked) ;
  2. Install Flash Player for Android 4.0+ or Flash Player for Android 2 and 3 (once the download is finished, open your notifications bar and tap the file to directly install it) ;
  3. Use a Flash-compatible browser (Firefox, Dolphin Browser).

Frequently Asked Questions

What and how do you emulate from the game?

The script finds in the page all the information used by the Flash game it can process (that is, not everything). In the Room tab, you can see the equipments, the crewmembers and the inventory of the room, you can move, and you should be warned if the room is on fire. Without Flash, you cannot see the Flash minimap, nor see nor aim for the hunters you can see from a patrol ship or a turret. In patrol ships, you can still shoot randomly from the Patrol command equipment, that may help.

Why is the Flash game not fully zoomed?

I cannot do anything to improve this, but it should be enough to play.

How do I know if a camera is watching me?

An installed camera is considered an equipment by the game, thus you will find it in the Room tab.

Is it normal that the game is long to refresh?

Unfortunately yes, a phone is less powerful thus slower than a computer for this task. The chat is the heaviest part, so you can get rid of some old faves or reopen a private channel when it's getting old to make it a bit faster. You can also unload the chat from the script parameters to lighten the main channel, in the same way as the Ctrl+W script.

Does it consume a lot of data to play Mush on a phone?

As long as you don't wipe your cache every minute, not really. The script only changes when I update it, so the only thing your phone will download is the page itself: 50 to 100 Ko, reloaded at every action. This is about twice lighter than a Google search.

How are the dorm beds sorted?

From the front to the rear of the ship. If you want the teddy bear, lie in the Bravo 1 bed :P Note though that the bed images in the Room tab are chosen at random, because I cannot know who lies in which bed.

One of my messages was posted in the wrong channel!

This bug is not caused by the script (I experienced it once, long before I started working on this), that is a problem that may (rarely) happen with mobile browsers. I tried to fix it, but it is very hard to reproduce. Still, I recently added the feature of displaying which channel the game believes you're in, but I don't know if that will help.
